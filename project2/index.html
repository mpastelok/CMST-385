Project Proposal

Project Concept

This project involves the creation of a professional portfolio website as a visual designer. The website will showcase projects I created using Adobe Photoshop, with a focus on strong composition, mood, and visual storytelling. The portfolio will serve as a centralized space to present my creative work in a clean, organized, and engaging way.

The primary objective of the website is to present my visual design work in a professional manner that highlights both my technical skill and creative intent. The goal of the website is to spark curiosity and inspiration in visitors, encouraging them to explore individual projects and ultimately reach out for employment opportunities.

The intended audience for this website includes potential employers and clients who are interested in creative services. These viewers may be seeking examples of past work, insight into my creative approach, or a way to make contact.

The website will include the following key features:

  • A clear and consistent navigation at the top of all pages.
  • A Home page that introduces me and highlights my selected projects.
  • Individual project pages that allow visitors to focus on one project at a time.
  • An About Me page that provides brief background information.
  • A Contact page with a contact form and other communication options.

Site Map

  1. How the website content will be organized:
    1. The website is organized around a central Home page, which functions as the hub. From the Home page, visitors can access all other pages either through the top navigation menu or interactive buttons for projects.
  2. Sub-topics included on the website:
    1. My Work (Project 1 and Project 2)
    2. About Me
    3. Contact
  3. Pages included on the website:
    1. Home (index.html)
    2. Project 1 (project1.html)
    3. Project 2 (project2.html)
    4. About Me (about.html)
    5. Contact (contact.html)
  4. Page layout and navigation:
    1. The top navigation bar will include My Work +, About Me, and Contact buttons on the left side, with my name centered, and a social link on the right-hand side. My Work and name will work as home buttons, with My work having hover-over dropdown links to Project 1 and Project 2. The home page will also feature buttons for Project 1 and Project 2. Navigation and footers will appear consistently on all pages, with a centralized layout used throughout the website.

Project Resources

W3Schools (HTML and CSS tutorials)

W3Schools will be used as my main reference for proper HTML5 structure and CSS implementation. This resource will help ensure the website follows correct syntax and validates without errors.

How to create a CSS navigation bar in 6 minutes! (YouTube tutorial)

A video tutorial on creating a CSS-based dropdown navigation menu will be used to help implement the “My Work” hover menu that link to individual project pages. This resource will assist with understanding hover states, layout, and acceptable positioning.

Adobe Photoshop

Adobe Photoshop will be used to edit and optimize all project images for web use. Images will be resized and compressed to meet requirements while maintaining visual quality and consistency throughout the website.
